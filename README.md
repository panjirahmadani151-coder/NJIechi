# NJIechi
Web toko baju sederhana dengan starter front-end untuk NusantaraMall.

## Cara menjalankan
1. Buka file `index.html` di browser.
2. Atau jalankan server lokal dari root workspace:
   - `python3 -m http.server 8000`
   - buka `http://localhost:8000`

## Apa saja di dalam project
- `index.html` — halaman utama toko baju.
- `styles.css` — gaya visual responsif.
- `app.js` — logika produk, keranjang, pencarian, dan checkout sederhana.
- `NusantaraMall_Technical_Specification.md` — dokumen spesifikasi teknis platform.
